# Отчет об обнаруженных дефектах

**Окружение**: `https://qa-internship.avito.com`, `postman collection`

## Список дефектов

### 1. Создание объявления с отрицательными числовыми значениями

**Краткое описание**  
Система позволяет создавать объявления, в которых числовые поля имеют отрицательные значения

**Шаги воспроизведения**

    1. Подготовить структуру к запросу: 
    {
          "sellerID": "<integer>",
          "name": "<string>",
          "price": "<integer>",
          "statistics": {
            "likes": "<integer>",
            "viewCount": "<integer>",
            "contacts": "<integer>"
          }
    }
    2. Выбрать одно из числовых полей (price, sellerID, viewCount, contacts) и присвоить ему отрицательное значение
    3. Отправить POST-запрос на /api/1/item для создания объявления
    4. Проверить статус-код ответа и факт создания объекта в системе

**Фактический результат**

    1. Сервер возвращает статус 200 OK
    2. Объявление с отрицательными значениями успешно сохраняется

**Ожидаемый результат**

    1. Сервер возвращает статус-код 400 Bad Request
    2. Система не дает создать объявление с отрицательным значением в полях price, sellerID, viewCount, contacts

**Серьёзность**  
Критическая

### 6. Неправильный статус код в теле при обращении к несуществующему роуту

**Краткое описание**  
Система отправляет в теле ответа по несуществующему роуту `status: 400`, в ответе же приходит статус `404`

**Шаги воспроизведения**

    1. Отправить POST запрос на несуществующий роут, например /api/6/not_found
    2. Проверить статус-код ответа и содержимое ответа

**Фактический результат**

    1. Сервер возвращает статус 404 NOT FOUND
    2. В теле ответа возвращается статус 400

**Ожидаемый результат**

    1. Сервер возвращает статус-код 404 Not Found, в теле ответа также отобращается статус 404

**Серьёзность**  
Низкая

### 7. Неправильное отображение моделей ответа в postman библиотеке

createdAt, но система возвращает после создания только статус, внутри которого id

**Шаги воспроизведения**

**Фактический результат**

    1. Сервер возвращает только поле status, cодержащий в себе id объявления

**Ожидаемый результат**

    1. Коллекция постман унифицируется с сервисом или ручка создания объявления меняет возвращаемую структуру

**Серьёзность**  
Низкая
